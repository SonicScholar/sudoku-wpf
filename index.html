Sudoku WPF

A simple .NET/C# command line app that can solve Sudoku puzzles. (Working on making a WPF GUI)

I made an engine that solves sudoku puzzles. Very rudimentary. However, I believe it can solve all Sudoku puzzles that can be solved without making a guess. I haven't put any "guess" algorithms in

Collaborators Welcome!

This is really a hobby project in an attempt to familiarize with .NET and WPF. Sharing expertise via collaboration is more than welcome with this project

Next steps/Feature Ideas - help welcome!

  • Display SudokuBoard object in some kind of WPF Grid
  • Add the ability for a human to change values
  • show cell candidates in empty cells
  • show "hints" on cells that have candidates that can be removed, and which algorithm gives the hint
  • Improve csv format of sudoku puzzles
  • Add a puzzle generator to the Sudoku Engine

Documentation = Very No

It would be nice to have this project be a tutorial of sorts, so there may be some nice documentation in the future.
